HTML

  • 纯html+css实现下拉菜单

    						此处以ul为例
						首先需要写个ul
						
						<ul>
							<li><a>aaa</a></li>
							<li><a>aaa</a></li>
						</ul>
						
						之后嵌套一个ul在里面
						
						<ul>
							<li><a>aaa</a>
								<ul>
									<li>bbb</li>
								</ul>
							</li>
							<li><a>aaa</a>
								<ul>
									<li>bbb</li>
								</ul>
							</li>
						</ul>
						
						然后让最里面的ul的display:none;
						当你hover外层的li时让内层的ul的display:block;
						ok，这样就实现了，这个方法最难处理的是二
						层的位置问题，自己琢磨吧
